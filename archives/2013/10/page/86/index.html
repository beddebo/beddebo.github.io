  • MySQL

  • sqlite

  • PostgreSQL

  • 微软SQL Server

  • Microsoft Access


通过仅提及你的登录细节。在那之后,你可以自定义生成的报告通过选择任意列的表和应用过滤器。



可选地,可以导入数据从Excel电子表格、ODS电子表格、CSV文件和谷歌Docs电子表格。


创造美丽的和交互式的沙发HTML输出(由JavaScript)。同时,它有表格输出,可以直接打开在MS Excel或粘贴到OpenOffice钙。
