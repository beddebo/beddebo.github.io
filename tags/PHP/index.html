简单PHP代理是一个PHP脚本,它旨在填补这一漏洞,以创建一个层之间的JavaScript代码和远程域获取数据在HTML,XML或JSON格式。


一些功能简单的PHP代理:



  • 请求的url可以对那些列入“白名单”&验证正则表达式。

  • 可选地向前客户机cookie / SID到远程服务器。

  • 可选地向前可配置的用户代理到远程服务器。

  • 请求可以使用GET或POST请求方法。

  • 远程数据可以按原样传递与所有远程头完好无损。

  • 远程数据可以包装在一个JSON / P结构,包括状态代码和远程头。

  • 如果使用JSON / P &远程数据是有效的JSON,它将被并入产生的数据对象。
