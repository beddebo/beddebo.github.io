jquery ahm:单行Ajax请求

jquery ahm是一个非常小的(~ 1 kb)jQuery插件使我们创建Ajax请求与更小的代码段。


它实际上是一个包装器的流行 $.ajax方法和带来一种不同的方法来处理回调。


而不是创建自定义回调为每一个Ajax请求,jquery ahm让端定义Ajax回调(自动处理)。



这里是一个例子,所有的回调处理在服务器端:



<?php
$response = array(
    // $("#animate").fadeOut("slow").delay("1000").slideDown("fast")
    "#animate/fadeOut"   => "slow",
    "#animate/delay"     => "1000",
    "#animate/slideDown" => "fast",
);
echo json_encode($response);
?>

坏的一面是,你移动前端代码和功能到服务器端。这不是首选方法许多设计师/开发人员和主要困难,如果前面和后端开发人员不同的人。


和很好的一部分,前端代码的大小可以减少严重,特别是网站的很多Ajax交互和你得到很多的灵活性在管理回调。
