免费的MySQL管理工具为Mac OS X:Sequel Pro

Sequel Pro是一个免费的MySQL管理工具为Mac OS X。


它提供了全表管理(包括索引),支持MySQL视图、进口/出口从SQL / CSV文件&兼容MySQL 3/4/5数据库。



应用程序支持SSH隧道连接除了标准的本地和远程连接达到数据库防火墙背后。


它可以连接到多个数据库的同时,提供了自动语法突出您的自定义查询,支持~ 30种不同的编码选项&更多。


备注:这个项目是开源&你可以检查源为进一步提高。


要求:
Mac OS X 10.5 +


网站:
