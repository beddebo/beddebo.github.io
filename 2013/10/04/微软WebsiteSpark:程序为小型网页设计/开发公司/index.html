微软WebsiteSpark:程序为小型网页设计/开发公司

很容易猜到,大多数网页设计师/开发人员不是球迷的微软。


另一方面,最近,他们经常为社区创造有趣的东西像WebsiteSpark.


WebsiteSpark不是一个软件,相反,它是一个程序,目标小网页设计公司为他们提供有价值的软件及点燃他们的业务。



程序没有前期成本3年&需要不同数量的支付取决于您想要使用的产品在这一时期。同时,您可以随时退出程序仅支付100美元的费用,这仍然是acceptible。


这个软件包括:




  • 3许可证的Visual Studio 2008专业版

  • 1许可证的表达工作室3(其中包括Expression Blend,Sketchflow和Web)

  • 2许可的Expression Web 3

  • 4处理器许可证的Windows Web Server 2008 R2

  • 4处理器许可证的SQL Server 2008网页版

  • DotNetPanel控制面板(允许方便远程/托管管理你的服务器)


对于任何愿意或已经使用微软平台创建网站,WebsiteSpark绝对是一个机会去看看。


提示:这个项目需要一个推荐计划,可以从网络获得合作伙伴。如果你没有,你可以电子邮件webspark@microsoft.com获取一个推荐码很快。
