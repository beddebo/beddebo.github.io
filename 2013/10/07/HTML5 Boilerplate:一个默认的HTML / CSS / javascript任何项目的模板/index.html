HTML5 Boilerplate:一个默认的HTML / CSS / javascript任何项目的模板

几天前,HTML5复位。和,这是一个新的选项:


HTML5 Boilerplate是一个HTML / CSS / javascript默认模板(起点),有助于网站待建工作跨浏览器支持html5 css3和快速工作。


模板都具有不同的性能优化和可选的功能,比如跨域Ajax和烦恼。



一个。htaccess配置文件包括缓存规则和准备网站为HTML5视频,使用下,启用gzip。


这也是建立与移动浏览器考虑和有一个iOS、Android、歌剧移动适应性标记+ CSS框架。


HTML5 Boilerplate提供其他许多功能,比如一个最优的打印样式表,一个测试适合单元测试和更多。


兼容性:
所有主流浏览器
