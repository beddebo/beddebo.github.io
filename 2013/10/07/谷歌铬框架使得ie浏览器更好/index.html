谷歌铬框架使得ie浏览器更好

谷歌铬框架是一个开源的Internet Explorer插件,使IE使用开放网络技术和快速的JavaScript引擎谷歌Chrome。


它实际上是不太新鲜,在周围转了一会,然而,稳定版本的插件是最新发布的和它是现成的。



插件需要2个步骤的工作:



  • 用户与IE 6、7或8应该安装它

  • 该网站必须添加一个标签到web页面(检测IE用户和插入标记只对他们是一个好主意)


在那之后,任何用户和浏览器IE版本将有一个更好的功能像支持HTML5,CSS3,更好的性能和更。JS


谷歌铬框架已经采用了许多流行的网站包括DeviantArt,github或HootSuite。


这绝对不是一个完美的解决方案是“一个步骤从最终用户是必需的”但至少向前迈进的一步,几乎不增加额外的努力对于网页设计师/开发人员。


要求:
Windows 7 / Vista / XP SP2


兼容性:
IE 6、7和8


网站:
http://www.google.com/chromeframe/
