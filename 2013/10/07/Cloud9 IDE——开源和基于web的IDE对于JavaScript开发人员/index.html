Cloud9 IDE——开源和基于web的IDE对于JavaScript开发人员

Cloud9 IDE是一个开源应用程序,旨在提供一个强大的和集中的IDE对于JavaScript开发人员。


它是由Ajax.org平台+利用HTML5,节点。js、插座。io和许多其他项目。


应用程序有一个非常快的文本编辑器和捆绑语法高亮显示支持javascript、HTML、CSS和混合模式。



Cloud9 IDE已经集成调试器进行节点。js和谷歌Chrome可以开始、暂停和停止从IDE。


使用WebDAV,它可以连接到远程工作区也一个插件系统进一步改进(已经有几个捆绑插件)。


备注:应用程序仍然是在早期发展阶段,可能有点车但continiously改善。


要求:
Git 1.7 +(对于Windows安装)


网站:
http://www.cloud9ide.com/


下载:
http://github.com/ajaxorg/cloud9
