沙发,统计、分析和报告任何数据

沙发是一个开源的统计、分析和报告应用程序,可以收集数据从一个选定的源。


它不强迫你使用离线数据和可以连接到各种生活数据库包括:



  • MySQL

  • sqlite

  • PostgreSQL

  • 微软SQL Server

  • Microsoft Access


通过仅提及你的登录细节。在那之后,你可以自定义生成的报告通过选择任意列的表和应用过滤器。



可选地,可以导入数据从Excel电子表格、ODS电子表格、CSV文件和谷歌Docs电子表格。


创造美丽的和交互式的沙发HTML输出(由JavaScript)。同时,它有表格输出,可以直接打开在MS Excel或粘贴到OpenOffice钙。


如果小事情改变你的数据集,您不需要重新导入数据,但是他们可以手动编辑。


有许多现成的报告像行和列的百分比、平均数、中位数、标准差、min - max和更多。


兼容性:
Windows,Mac,Linux发行版Ubuntu和其他
