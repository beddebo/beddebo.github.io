Geo启用网站与JS:YQL Geo图书馆

地理位置是最受欢迎的学科之一,对于今天的web &有许多解决方案,帮助geo启用web应用程序。


YQL Geo图书馆是一个令人印象深刻的资源地理定位,因为这是完全自由和javascript。


它有各种强大的功能,例如:



  • 检测访问者的位置 W3CGEO API和IP作为后备

  • 发现地理位置从文本

  • 找到位置的纬度/经度一对

  • 找到位置在一定网络文档(通过 URL)

  • 得到位置一定IP号码



图书馆是轻量级(4 kb未压缩的)&非复杂。它只是通过调用函数:



yqlgeo.get(‘217.12.14.240’,function(o){
  alert(o.place.name + ‘,’ + o.place.country.content +
        ‘ (’ + o.place.centroid.latitude + ‘,’ +
               o.place.centroid.longitude + ‘)’
        );
});

能把你的城市、乡村、纬度和经度的任何给定的IP。
