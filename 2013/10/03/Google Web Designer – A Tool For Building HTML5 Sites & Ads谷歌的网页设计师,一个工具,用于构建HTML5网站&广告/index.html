Google Web Designer – A Tool For Building HTML5 Sites & Ads谷歌的网页设计师,一个工具,用于构建HTML5网站&广告


Google has released a new tool, named谷歌发布了一个新的工具,命名Google Web Designer谷歌网页设计师, for creating interactive HTML5 sites and ads.,为创建交互式HTML5网站和广告。


It is a modern WYSIWYG where you don’t need to dive into the code but use the features to get the output and the tools are mostly design-oriented.这是一个现代所见即所得,你不需要潜入代码但使用特性来得到输出和工具主要是设计型的。


However, the code generated is always there and it can be edited or tweaked and the result is displayed automatically.不过,生成的代码总是存在,它可以被编辑或修改,结果自动显示。



The tool comes with ready-to-use settings for designing Google-powered ads (DocubleClick, AdMob) that will work on any device.这个工具有现成的设置谷歌驱动的广告设计(DocubleClick,AdMob),将工作在任何设备。


There are built-in components like 360 or carousel galleries, YouTube video embedding and more.有内置的组件(比如360或旋转木马画廊,YouTube视频嵌入和更多。


A timeline exists for animations and anything can be drawn with a pen tool (not limited to shapes).存在一个时间轴动画和任何可以得出用钢笔工具(不限于形状)。


Google Web Designer is currently in beta status and it is available for both Windows + Mac.谷歌网页设计师目前在beta状态并可用于Windows + Mac。
