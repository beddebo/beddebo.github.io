Peerbind - JS API互联网站

Peerbind是一个JavaScript API(建造的A9亚马逊)实现P2P通信在一个网站几乎立即。


它通过添加一个事件监听器来触发事件的元素共享所有游客在peerset(游客web页面)。


简单,它提供了一个非常易于实现实时、交互方式网络通信(聊天、多人游戏,文档共享,等等)。



API是一个jQuery插件与方法来管理事件和peerset创建(它可以缩小到用户提供相同的IP,地理位置、领域或一个字符串——比如一个秘密密钥)。


有一个免费的公共服务器提供,有利于检测和底层应用程序。


和,服务器端JS提供亚马逊许可证(这应该是托管在Amazon AWS)为一个更稳定的和定制的用法。


网站:
http://peerbind.com/
