增强的2 d Flash API -八哥框架

几天前,Adobe Flash Player 11已经启动了这是一个里程碑,该产品作为新Stage3D API,允许多达1000��更快的渲染而Flash Player 10。


它侧重于3 d内容和广泛使用的2 d内容不会得到这个新特性的优点。



八哥框架,一个纯粹的ActionScript 3库,旨在填补这一缺口,通过GPU渲染所有的内容。


框架有一个熟悉的API,这使得它很容易使用为Flash开发者。甚至可以提高生活的Flash网站与它只是通过交换本机闪光类与八哥等价物。
