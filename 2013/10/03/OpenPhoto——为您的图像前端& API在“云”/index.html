OpenPhoto——为您的图像前端& API在“云”

许多互联网用户存储他们的图片照片共享/托管服务,如Flickr,SmugMug或Picasa,确保它们很容易达成,而且定期备份。


OpenPhoto是一个开源的PHP应用程序,鼓励用户主机的图像在云(Amazon S3,Rackspace CloudFiles,谷歌存储等)在自己的和功能作为一个前端管理+呈现它们。



这样,图像保持完全控制下的用户,切换供应商越来越便捷和用户不会被限制的功能提供照片共享/托管服务。


前端有一个简单的界面,照片显示为一个网格,分类与标签和搜索很容易。这也是移动并进行主题化支持。


OpenPhoto本身是建立在顶部的强大的API,这使得它可以改进或与其他应用程序集成的照片。
