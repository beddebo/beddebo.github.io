SQLyog是什么?


这是一个桌面应用程序来管理任何数量的MySQL数据库,运行在任意数量的主机。


SQLyog处理创建/编辑和删除数据库、用户、表和记录像微风。



应用程序可以导入/导出数据(CSV / XML / HTML / Excel)和备份或恢复数据库/表。它甚至可以运行这些命令(或任何其他命令/查询)作为调度的任务。
