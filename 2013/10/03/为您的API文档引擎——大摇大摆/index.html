为您的API文档引擎——大摇大摆

如果你有一个web应用程序与一个API,一个好的文档使它更容易为用户使用它和帮助服务增长更快。


大摇大摆是一个开源的应用程序,它支持基于rest的api,同时解决了服务器、客户端和文档/沙箱的需要。


它使开发人员和非开发人员可以与API在沙箱的UI,给了一个好主意在API的方式响应参数和选项。



它讲JSON + XML和附带实现在Scala中,Java和HTML5。,客户端发电机可用于Scala,Java、Javascript、红宝石、PHP和Actionscript 3。


它是由一个工作实现Wordnik和能找到Wordnik的API文档页面.
