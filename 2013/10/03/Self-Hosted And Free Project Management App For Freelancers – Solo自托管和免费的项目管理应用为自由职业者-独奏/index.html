Self-Hosted And Free Project Management App For Freelancers – Solo自托管和免费的项目管理应用为自由职业者-独奏


Solo独奏is a free project management application that is built with freelancers in mind.是一个免费的项目管理应用程序,是建立与自由职业者在心里。


It is self-hosted,聽created with PHP and uses MySQL for storing the data.这是自托管,聽创建PHP和MySQL存储数据的使用。


The application has a slick interface where a single user (limited to a user) can create any number of projects + tasks and bind them with clients.应用程序有一个光滑的界面,一个用户(限于用户)可以创建任意数量的项目+任务和绑定他们的客户。



Tasks can be easily searched, reordered with drag ‘n’ drops and files can be attached to them.任务可以轻松地搜索、重新排序与拖动“n”滴和文件可以被附加到他们。


File uploader is also powerful, it can handle multiple files and has inline file previews.文件上传者也是强大的,它可以处理多个文件和已内联文件预览。


There is a project calendar to view tasks quickly, an activity stream shows all the footprint of yours and a timer helps you keep track of the time spent for each task/project/client.有一个项目日历查看任务很快,活动流显示所有你的足迹和一个计时器可以帮助你跟踪的时间为每个任务/项目/客户端。
