LiteCartLitecartis a free and open source shopping cart application that is built with PHP-MySQL.是一个免费的、开源的购物车应用程序,是建立与php mysql。


It is light, not in means of features but for the usage (less clicks, improved usability..) and for development.它是光,而不是意味着功能但用法(更少的点击,提高可用性. .)和开发。



All the main features of an e-store is offered as a standard and more stuff exist like: product groups, different prices per currency, automatic backups and more.所有的主要功能是提供销售的标准和更多的东西存在如:产品组,不同的价格,自动备份和每货币更多。


And, there is an easy-to-use control panel for setting up the store and managing everything about it.并且,有一个易于使用的控制面板设置存储和管理有关它的一切。
