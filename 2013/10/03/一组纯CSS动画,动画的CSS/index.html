一组纯CSS动画,动画的CSS

动画像淡入/淡出或弹性都使用几乎无处不在,十分方便,JS框架提供,创造有吸引力的用户体验。


许多这些动画实际上有等价物在CSS和css动画减轻他们的用法。


它是一个CSS文件,包括各种规则如flash动画,弹跳,动摇和多个褪色,旋转+反弹。


每个动画是一个 class。像CSS动画有本机硬件加速,他们的工作更为顺畅比JS动画。然而,这将是一个良好的实践作为后备。JS动画
