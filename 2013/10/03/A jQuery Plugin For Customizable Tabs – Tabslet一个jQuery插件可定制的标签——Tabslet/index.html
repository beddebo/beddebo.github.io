A jQuery Plugin For Customizable Tabs – Tabslet一个jQuery插件可定制的标签——Tabslet


There are various jQuery-powered solutions for creating tabs quickly and easily.有各种各样的jquery开发解决方案快速轻松地创建选项卡。


TabsletTabsletis another alternative for the task and has support for multiple handy features.是另一个替代的任务,并支持多种方便的功能。



It simply converts an unordered list into a tabbed interface and has callbacks on major events (when the tab is clicked or the tab content is loaded).它只是一个无序列表转换成一个选项卡式界面,并在重大事件回调(当选项卡单击或选项卡内容加载)。


Also, tabs can be rotated automatically on given intervals and their content can be displayed with an animation.同时,标签可以旋转自动给定间隔和他们的内容可以显示为一个动画。
