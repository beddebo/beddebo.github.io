响应与jQuery UI - rlightbox灯箱

rlightbox是一个lightbox插件的基础上构建jQuery + jQuery UI和带有一些新鲜的特性,并不存在于它的替代品。


它可以显示任何类型的内容从图像视频(YouTube和Vimeo),Flash文件和HTML。


插件有一个独特的“全景”功能只显示一个图像的一部分,但允许用户查看其余与平移。



���,“生活调整“汽车尺度图像按照浏览器大小(这是伟大的响应性设计)。


rlightbox可以设置为显示一个项目或一组的ehelp与th预下按钮(有支持键盘导航)。


这是外观和感觉可以完全定制的插件是完全兼容解压缩ThemeRoller jQuery UI。


要求:
jQuery和jQuery UI


兼容性:
所有主流浏览器
