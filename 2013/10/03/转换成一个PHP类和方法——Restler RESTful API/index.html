转换成一个PHP类和方法——Restler RESTful API

restler是一个多协议和开放源码框架PHP类和方法公开为一个REST API。


这个框架是轻量级(只能通过几个文件)和开发人员可以使用任何有经验的面向对象的PHP没有学习曲线。


它使用多种格式:XML、JSON和AMF Plist。同时,它非常模块化这意味着创建新的格式或功能是可能的。



Restler可以转换格式用两种方式(请求和响应)和支持GET、POST、PUT和DELETE HTTP请求。
