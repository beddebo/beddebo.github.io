链接的支持——chainvas Canvas API

chainvas是一个轻量级(1 kb大小)的JavaScript库,尝试建立 <canvas>API若(像那些在jQuery,MooTools,等等)。和,也适合任何JavaScript api。


它增加了方法的原型 CanvasRenderingContext2D和替换现有的包装器调用原始的函数并返回 this如果,只是如果函数没有返回 undefined.


还有额外的辅助方法的属性设置在一个可读的格式,画圆圈和圆角矩形。



要求:
没有要求


兼容性:
所有现代浏览器
