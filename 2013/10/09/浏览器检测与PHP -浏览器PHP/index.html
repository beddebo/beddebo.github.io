浏览器检测与PHP -浏览器PHP

学习用户的浏览器及其版本通常是非常重要的决定如果任何其他脚本和样式需要插入到web页面。


和,这些任务通常是通过JavaScript作为我们不需要去那么深(检测如果是IE或不是通常是足够的)。



浏览器php是一个非常有用的PHP类,帮助收集数据访问者的浏览器包括:



  • 平台使用

  • 浏览器

  • 浏览器版本


它可以检测几乎所有流行的浏览器包括手机+平板设备和机器人用几行就像:




$browser = new Browser();
if( $browser->getBrowser() == Browser::BROWSER_FIREFOX && $browser->getVersion() >= 2 ) {
    echo ‘You have FireFox version 2 or greater’;
}


如果需要浏览器检测不仅仅是分析是否IE、Firefox、Chrome或Safari浏览器,然后。php是绝对必须的书签。


要求:
PHP
