ProcessWire是一个开源CMS应用程序,是建立与PHP和简约。


应用程序有一个非常低的学习曲线没有复杂性和它是漂亮的开发者/设计者友好考虑它是建立适应你的做事的方式。

和,因为最大的灵活性,字段可以很简单,比如包含原始文本或数字,或更复杂的像富文本编辑、页面之间的关系和字段保存文件或图片。
