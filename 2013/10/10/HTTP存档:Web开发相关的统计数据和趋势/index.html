HTTP存档:Web开发相关的统计数据和趋势

对于网页设计师/开发人员,拥有一个详细的了解整体与web相关的统计数据是相当有价值的更好地分析你要做的是正确的(好)或不是。


http存档是一个网站,适用于定期检查17000 +网站,收集信息和输出web开发相关,最新的报告。



报告分为趋势和统计数据,趋势包括:



  • 平均响应时间

  • HTML / CSS / JS /图像大小和请求的数量

  • 页面速度得分

  • 和更多. .


和统计数据包括:



  • 流行的JavaScript库

  • 流行的脚本

  • 页面使用谷歌库API

  • 页面使用闪光灯

  • 和更多. .


HTTP归档是一个开放源码项目,您可以轻松地执行一个自定义的安装和运行更频繁的检查,创建新报告和/或获得统计数据为一组不同的网站。
